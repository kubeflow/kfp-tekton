# Security Policy

## Supported Versions

Below are the list of supported KFP-Tekton for security and bug fixes.

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.8.x   | :white_check_mark: |
| 1.7.x   | :white_check_mark: |
| < 1.7   | :x:                |

## Reporting a Vulnerability

Please summit [an issue](https://github.com/kubeflow/kfp-tekton/issues) in the KFP-Tekton repo for any vulnerability you found in this project.
